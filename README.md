# Secret Santa Wishlist 🎁

A Django web application that helps friends, families, and groups organise Secret Santa gift exchanges with less chaos and more fun.

Users can create groups, join existing groups, build wishlists, and manage gift exchange planning all in one place.

## 🚧 Work in Progress

This project is still actively being developed and improved. New features, design updates, and bug fixes are being added regularly.

## ✨ Features

- User registration and login system
- Create and join Secret Santa groups
- Personal dashboards
- Wishlist creation and management
- Add, edit, and delete wishlist items
- View group members
- Responsive Bootstrap UI
- Modal-based forms for smoother user experience

## 🛠 Built With

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite (development)

## 📌 Purpose

The goal of this project is to make organising Secret Santa gift exchanges easier, cleaner, and more enjoyable for families, friends, and workplaces.

Instead of messy group chats and random spreadsheets, everything is managed in one place.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/alex-enc/secret-santa-wishlist-django.git
cd secret-santa-wishlist-django
```
### 2. Create a virtual environment
```bash
python -m venv venv
```
### 3. Activate the virtual environment
#### Windows
```bash
venv\Scripts\activate
```
#### Mac/Linux
```bash
source venv/bin/activate
```
### 4. Install dependencies
```bash
pip install -r requirements.txt
```
### 5. Run migrations
```bash
python manage.py migrate
```
### 6. Start the development server
```bash
python manage.py runserver
```
